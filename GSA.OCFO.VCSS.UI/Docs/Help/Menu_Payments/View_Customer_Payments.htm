View Customer Payments

The payments view page provides a list of the payments made against a given customer account.  

 

 

The top of the screen has a set of search criteria that can be employed to find particular payments.  None of the search criteria is mandatory.

 

After the search:

 

  • Select a line item and click the View button to see a more comprehensive list of a payment transaction's fields.
  • Select a line item and click the View Statement button to jump to a view of the statement to which the payment pertains.
  • Click the Account Summary button to jump to the Account Summary screen.
  • Click the Sort button to sort the list on up to three fields in the list.
  • Click the View as CSV button to download the search results to a comma separated values (CSV) file.
