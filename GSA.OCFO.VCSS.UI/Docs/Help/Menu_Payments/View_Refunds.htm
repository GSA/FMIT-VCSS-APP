View Refunds

Select view refunds to see a list of refunds applied to an account.

 

 

All of the search criteria on this page are optional. Enter criteria if desired, and then click the Search button. When the list of refunds appears:

 

  • Select a line item and click View to see more details about the refund.
  • Click the Sort button, to sort the payments list on up to three fields in the list.
  • Click the View as CSV button, to download the search results to a comma separated values (CSV) file.
