Create Statement Correspondence

Use Create Statement Correspondence to send a message to the GSA about a statement.

 

 

 

Enter a statement number, or click the "Billing Statement" link to search for the statement number to which the new correspondence will pertain, and then click "next".

 

 

On the final screen fill in the required fields, and any optional fields desired.

 

Attach supporting documentation, if desired, by clicking the Attachments tab.

 

Click Submit Correspondence
