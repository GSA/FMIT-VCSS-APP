Outstanding Balances by Account

The Outstanding Balances by Account page provides the outstanding balance information for the accounts to which the user has access.

 

 

The page displays one line item per account code. Select one of the line items by clicking the check-box next to it, and do one of the following:

 

  • Click on the View Outstanding Statements button to jump to the Statement Search page.  See the View and Print Statements topic for more information on the Statement Search page.
  • Click on the View Recent Statements button to see a list of the statements that have line items bearing that account number that were processed within the last three months.   
  • Click on the Send Correspondence button to send a new correspondence record to GSA pertaining to the selected account code.  See the Create Account Correspondence topic for more information on creating account correspondence.
  • Click the Sort button and select up to three fields on which to perform a sort.

  • Click on the View as CSV button and download a comma separated variable file containing the data in the search results frame of the screen.
