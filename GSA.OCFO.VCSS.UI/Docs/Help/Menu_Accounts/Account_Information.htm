Account Information

The Account Information page displays information about an Account Code.

 

If a user has access to more than one Account Code, they should select the Account Search menu option and search for and select an account code to reach the Account Information page.  If a user has access to only one Account Code, they should select the "Account Information" menu option to reach the Account Information page.

 

 

 

 

The account information screen has three tabs:

 

  • The account information tab -  information such as the Account Code, name, agency, and bureau.

  • The address information tab - addresses such as the mailing address.

  • The users tab - the users with access to the Account Code's data.

 

 

Please inform the GSA, (using the VCSS Account Correspondence feature), of any issues with the data displayed on the account information or address information pages.
