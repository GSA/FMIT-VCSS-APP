Account Summary

The Account Summary page displays summary totals for the transactions bearing the account codes to which a user has access.

 

 

Enter search criteria (which must include a statement date range of one year or less), and click the search button.

 

When the results return:

 

  • Click on the View Statements button to go to the Statements Search page.
  • Click the View Payments button to go to the Payment Search page.
  • Click the Sort button to sort the returned results.
  • Click the View as CSV button to download the search results to a comma separated values file.
