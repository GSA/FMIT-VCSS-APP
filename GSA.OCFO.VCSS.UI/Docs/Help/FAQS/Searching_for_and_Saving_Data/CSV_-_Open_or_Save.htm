CSV - Open or Save?

The default behavior of spreadsheet packages like Microsoft Excel results in the loss of leading zeroes in text fields consisting of numeric characters, such as some account codes and vehicle tags.  Further, very long text values consisting of numeric characters are changed into scientific notation numbers.

 

To avoid this, SAVE the CSV file, and use the text import wizard of the spreadsheet software to import the CSV file.  During the import, tell the spreadsheet package which fields should be treated as text rather than as "general" values.
