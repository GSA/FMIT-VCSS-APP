How many search results does the item collection field hold?

The item collection field in a VCSS web browser page is a list of the results returned from a search of the database.  Due to .html and other limitations the system has been set to only return 999 rows of data when a search is performed.  If more records meet the search criteria used, the first 999 will be displayed on the web page and a warning message will appear.  

 

 

For all VCSS item collections other than that on the Statement Details page, the "View as CSV" button will generate a CSV file based on the item collection content.  So the CSV file for item collections where more than 999 rows met the search criteria entered will consist of the 999 rows in the item collection.

 

For the Statement Details page, the "View as CSV" button will cause VCSS to perform a new query, and the search results will contain all of the results returned.  However, note that there is a time limitation on queries performed in VCSS to avoid "runaway queries", so there is still a limitation, that being the number of rows that can be retrieved from the database in 15 minutes.  If the time to complete the query exceeds this, the results will still be only a portion of the search results.
