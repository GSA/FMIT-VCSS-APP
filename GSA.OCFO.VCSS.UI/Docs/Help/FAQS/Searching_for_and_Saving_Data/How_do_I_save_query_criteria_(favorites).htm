How can a user save query criteria (favorites)?

Frequently used search criteria can be saved as a favorite as described below.

 

To set up a favorite value:

 

Click the paperclip next to the field's name when on one of the search screens such as the Statement Details or View and Print Statements screens.  Select the "search" option from the pop-up window that appears.  Alternatively, click the underlined hyper-link, such as the "Account" hyper-link in this example.

 
 

On the next screen, perform a search for the desired value.

When the search results return, click the icon next to the desired value to save it as a favorite.


 

Click the "select" button on the same screen to return to the initial screen.

 

 

To use a favorite value:

 

Click the paperclip next to the field's name, such as the Account field in the illustration.

 

 

A pop-up window will appear with the favorites previously established.

 

Pick one of the values on the list, or pick the "search" item on the list to search for a different code not on the favorites list and to optionally add it to the favorites list.
