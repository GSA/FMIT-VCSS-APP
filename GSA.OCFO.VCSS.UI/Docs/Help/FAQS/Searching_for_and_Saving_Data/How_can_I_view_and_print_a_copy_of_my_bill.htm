How can a user view or print a copy of a bill?

Use the "View and Print Statements" function to view a statement or obtain a PDF version of the statement for printing.

 

Select the View and Print Statements menu option.

 

 

Enter a statement date range (no more than 366 days) and any other desired criteria.  Click Search.

 

 

To view the statement's summary and detail information click "View".

 

 

To view the statement as a PDF file, if one is available, click "View PDF".  If no PDF is available for the selected statement an error message will be shown, otherwise a PDF will open containing a printable version of the statement's information.
