How can a user download data to a local hard drive?

On screens that have them, click the "View as CSV" button above the search results / item collection field to download the data associated with that field to a CSV file.   Note that for every screen other than the Statement Details screen the CSV will match the item collection.  In the case of the Statement Details screen the CSV file will have many more columns than the search results field, and will contain all of the rows that match the search criteria used to populate the search results field.

 

Browser tips:

 

Fire Fox users can download the csv files by clicking the View as CSV button.

 

Internet Explorer users must take some additional steps.  Please see Internet Explorer Download Issues.
