Are there any issues when downloading using Internet Explorer?

The Internet Explorer security feature, which causes an "info bar" to appear in which one must confirm that one does desire to download the file in question, can cause issues in VCSS because of a conflict with the VCSS system's code.  To avoid the issue:

 

Depress the Control Key.  Hold it down.

 

 

Click on View as CSV.

 

 

 

Click on Save.

 

 

Release the Control key as soon as the Save As dialog appears.
