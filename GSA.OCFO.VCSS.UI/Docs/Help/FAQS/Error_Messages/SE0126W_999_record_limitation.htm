SE0126W The number of results returned by your query is equal to or greater than the configured maximum result limit of 999.  The system will display records up to that limit.

The web page field holding the search results from a search in VCSS is limited to 999 rows.  The user must enter more search criteria to filter the list of returned data so that only 999 rows are returned.  In the case of the Statement Details screen, however, the user can download the data in the search results to a CSV file and view the file using their spreadsheet or database software.
