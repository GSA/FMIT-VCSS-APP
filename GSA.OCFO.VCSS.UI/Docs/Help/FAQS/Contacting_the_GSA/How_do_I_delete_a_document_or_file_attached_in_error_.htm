Deleting Attachments

 

Files that have been added to the list of files to be attached to a correspondence record when submitted can be removed from the list by:

 

  • Selecting the item in the list of files to be attached.
  • Clicking the "Delete" button.

 

Files cannot be deleted from a correspondence that has already been submitted.  If a file is submitted in error, contact the GSA to have the file removed, (via another correspondence record).
