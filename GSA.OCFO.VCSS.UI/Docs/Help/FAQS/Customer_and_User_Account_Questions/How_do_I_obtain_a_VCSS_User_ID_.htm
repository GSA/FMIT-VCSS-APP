How does a prospective user obtain a VCSS User ID?

A user id is issued to those individuals who do not have a user id, when they do one of two things:

 

  1. Request (and receive) access to the data associated with one or more Account Codes or IPAC ALC codes.
  2. Successfully register as the account manager for an Account Code or IPAC ALC.
