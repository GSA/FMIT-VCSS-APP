Obtaining Access _ ALC Example:

Users can search for an Account code to which to request access using:

 

  • the Account Code

  • the BPN associated with the code, if any

  • the ALC used to process the Account Code's IPAC transactions

 

Further, they can search, using the ALC,  for an ALC whose IPAC transactions they wish to request permission to view.

 

This topic covers the process of searching for an Account Code or Codes or an ALC using the ALC.

 

 

Go to http://vcss.gsa.gov and click on the "Request access...." link.

 

 

Select the "customer" radio button and click "submit".

 

 

Check the ALC/Name radio button and click Submit.s

 

 

 

Enter a search string in the ALC field and click "Search".   

 

 

 

The result is a list of account table entries, containing one entry per Account Code with an IPAC ALC value containing the entered string plus one entry per ALC matching the entered string.

 

  • To request access to the transactions for a given customer account code click the check box next to that account code record.  

  • To request access to the IPAC transactions bearing a given ALC click the check box next to the ALC's account code record entry.

  • Click Submit Request.

 

 

 

 

Fill in the name, email, and phone number fields and click "Submit to GSA".

 

 

If the request is approved the GSA will send an email to the requesting user. If the user requesting access does not have a user id, and the request was approved, the GSA will send an email with a user id, and a second email with a password.
