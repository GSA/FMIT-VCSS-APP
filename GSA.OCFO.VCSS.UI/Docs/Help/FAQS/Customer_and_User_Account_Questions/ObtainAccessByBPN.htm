Obtaining Access _ BPN Example:

Users can search for an Account code to which to request access using

 

  • the Account Code

  • the BPN associated with the code, if any

  • the ALC used to process the Account Code's IPAC transactions

 

This topic covers the process of searching for the Account Code using the BPN.

 

First, go to http://vcss.gsa.gov and click on the "Request access...." link.

 

 

Select the "customer" radio button and click "submit".

 

 

Enter a search string in the BPN box and click "Submit".

 

 

Enter a search string to find all the Account Codes with BPN codes that contain the search string and click "Search".  

 

For instance, entering "123" will find all the Account Codes that have the BPN codes "12345678", "41235678", and "45123678", among others.

 

 

 

Click the check box next to the desired account code or codes and then the "Submit Request" button.  Note that you can also, in this window, repeat your search using a different search string, if desired, rather than selecting an account code from the list.

 

 

 

Fill in the contact information fields, and click the "Submit to GSA" button.

 

 

 

If the request is approved the GSA will send an email to the requesting user. If the user requesting access does not have a user id, and the request was approved, the GSA will send an email with a user id, and a second email with a password.
