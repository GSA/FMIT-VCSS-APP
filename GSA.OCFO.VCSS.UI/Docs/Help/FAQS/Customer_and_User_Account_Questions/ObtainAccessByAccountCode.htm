Obtaining Access _ Account Code Example:

Users can search for an Account code to which to request access using

 

  • the Account Code

  • the BPN associated with the code, if any

  • the ALC used to process the Account Code's IPAC transactions

 

This topic covers the process of searching for the Account Code using the code itself.

 

 

Go to http://vcss.gsa.gov and click on the "Request access...." link.

 

 

Select the "customer" radio button and click "submit".

 

 

Check the Account Code/Name radio button and click Submit..

 

.

Enter a search string in the Account Code field and click "Search".  This will find all of the account codes containing the entered value.  For instance, "123" would find "000123000" or "000123" or "123000".

 

 

 

The system will list all of the account codes that match the entered search string.  This example assumes there is only one code matching the string "123456".

 

Click the check box or boxes next to the desired code or codes and then the "Submit Request" button.   

 

 

 

Fill in the name, email, and other fields on the next screen and click "Submit to GSA".  

 

 

If the request is approved the GSA will send an email to the requesting user. If the user requesting access does not have a user id, and the request was approved, the GSA will send an email with a user id, and a second email with a password.
