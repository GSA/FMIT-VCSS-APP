Account and ALC Access Requests

Users must request permission before they can view the data associated with an Account Code or an ALC.  The request will be reviewed by the "account manager" for the Account Code or ALC.  If approved, the GSA will enable the user's user id so that it can see the data where the Account Code is the customer organization's Account Code or the ALC is the IPAC "designated agent".
