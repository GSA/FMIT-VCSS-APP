View and Print Statements

Select "View and Print Statements" to view a statement and its details or to obtain a printable PDF file of the statement.  Note that not all statements have PDF versions.

 

 

To use this page

 

  1. Optionally enter one or more of these search criteria:
    • a single Statement number on which to search.
    • a statement type (IPAC or non-IPAC).
    • a business line (such as fleet or rent).
    • a statement date range.
    • a single account code.
    • an Agency Location Code (ALC).
    • A BPN number (do not use if the Account Information page shows that VCSS does not have a BPN for the account code).
  2. Click the Search button.
     

When the search completes:

 

  • Select one of the search results lines, by clicking its check box, and click "View", to jump to the View Statement page for the selected statement.
  • Select one of the search results lines, by clicking its check box, and click "View PDF", to open a PDF version of the statement.  You can then save the PDF or print it locally if you wish.
  • Sort the list of statements, based on up to three of the list's fields.
  • Click the "View as CSV" button, to obtain a comma separated variable (CSV) file of the search results.
